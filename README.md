
# IoT Sensor Data Analytics and Monitoring Dashboard

## Overview
This project analyzes IoT sensor data (temperature, pressure, speed) using Python and visualizes insights using Power BI.

## Features
- Time-series data analysis
- Data cleaning and preprocessing
- Anomaly detection using Z-score
- Interactive dashboard for monitoring

## Tools Used
- Python (Pandas, NumPy)
- Power BI

## Output
The dashboard helps detect abnormal sensor behavior and monitor system performance.
